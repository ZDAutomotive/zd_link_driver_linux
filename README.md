# zdlink_driver_linux

A public repo to release zd-link driver for linux.

## Install

```bash

chmod +x zdlink-install.sh

```

### compliant mode

By default it installs compliant driver.

```bash

zdlink-install.sh

```

### legacy mode

```bash

zdlink-install.sh --legacy

```

## Notes

ubuntu system driver would be backuped as lan78xx.ubuntu
