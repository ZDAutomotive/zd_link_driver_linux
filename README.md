# zdlink_driver_linux


## Install

```bash

chmod +x zdlink-install.sh

```

To install the driver for zdlink, this script will first set all the zdlink interfaces down.

### compliant mode

By default it installs compliant driver.

```bash

./zdlink-install.sh

```

### legacy mode

```bash

./zdlink-install.sh --legacy

```
